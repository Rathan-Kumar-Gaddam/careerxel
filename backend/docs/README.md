# Backend Documentation

## Examples Directory

This directory contains example configuration and data files that show the expected format for various content types.

### Files

- **blogPosts.example.json** - Example structure for blog post data with fields like slug, title, excerpt, category, publishedDate, etc.
- **pricingContent.example.json** - Example pricing plans structure with audience configuration, plan tiers (Free, Plus, Pro), pricing in multiple currencies, and feature groups.
- **resourcesContent.example.json** - Example resources/newsletter content structure with blog metadata and newsletter configuration.

## How to Use

Copy these example files as templates when setting up new content types or integrating with CMS. The actual content is served from the Strapi backend.

## Notes

- Example files are version-controlled for reference
- Actual production data comes from the Strapi CMS
- Do not modify examples unless you're updating the expected schema
